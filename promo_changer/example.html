Promo Changer

You can include the promo changer as many times as you want on the same page. Just add the following three lines in the HTML HEAD section.

<script type="text/javascript" src="promos/jquery.js"></script>
<script type="text/javascript" src="promos/promos.js"></script>
<link type="text/css" rel="stylesheet" href="promos/promos.css" />

And add the following line in the HTML BODY section.

<div class="promo-changer"></div>

Configuration Settings

The following table shows all the configuration settings that can be used to override the default settings

Configuration Key Data Type Default Value Options Description
php string promos/   Relative path to the php file to read the XML file.
effect string fade fade OR slide OR plain The transition effect used to switch promo images.
Plain is a hard change and transitionTime has no effect
stillTime integer 8000   The time in milliseconds (1000 is one sec) that the image will display
between transitions
transitionTime integer slow slow OR fast OR 300 The time in milliseconds (1000 is one sec) that the image will take
to do the transitions.
enableTabs boolean true true OR false Enable or disable the click tabs.
useTransitionOnTabClick boolean true true OR false When a tab is clicked, use the transition effect or not.
containerClassname string container   Class name used for the entire container (tabs and images).
imgContainerClassname string img-container   Class name used for the image container.
tabsClassname string tabs   Class name used for the tabs container.
tabClassname string tab   Class name used for each clickable tab.
tabActiveClassname string active   Class name used for the current active tab.

Other Examples of Configuration

Use the following configurations below to change the behavior of the effect. Edit line 13 in the file promos.js

Slide Effect

jQuery(function(){
    
    $('.promo-changer').promos({
        
        effect : 'slide'
    });
});

Slow Fade

jQuery(function(){
    
    $('.promo-changer').promos({
        
        transitionTime : 5000,
        useTransitionOnTabClick : false
    });
});

Quick Fade

jQuery(function(){
    
    $('.promo-changer').promos({
        
        stillTime : 3000,
        transitionTime : 'fast',
        enableTabs : false
    });
});

Slow Slide

jQuery(function(){
    
    $('.promo-changer').promos({
        
        effect : 'slide',
        stillTime : 12000,
        transitionTime : 4000,
        useTransitionOnTabClick : false
    });
});

Default Settings

jQuery(function(){
    
    $('.promo-changer').promos();
});
